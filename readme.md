# school/university management system


this project is a school/university management system that let you manage students , classrooms , and teachers. this system can add,edit,omit,save,and load data. it is very useful for didactic projects or simple managment of data . you can save your data on `data.json`. 

---

- structure:
    - class:
      - students:
        propertis of this class are name , family ,  ID , and courses
      - techers:
         propertis of this class are name , family , grade , ID , and courses
      - courses:
        propertis of this class are ID , name , units , and scores
      - classroom:
        propertis of this class are ID , name , students , teacher , and  course  

---
### features:
- manage students
- manage teachers with special id
- add and manage different classrooms
- match students and teachers to classrooms
- save and load data
